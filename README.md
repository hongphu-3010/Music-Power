# Music Power

## Live demo

Official website: [https://hongphu-3010.github.io/Music-Power/](https://hongphu-3010.github.io/Music-Power/)

## Main technology used

- SCSS, Swiper

## Feature

- Play, Pause, Random, Repeat, Choose a song, Choose playlist, Change volume, Mute, Unmute
